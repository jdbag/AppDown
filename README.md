# AppDown Store

متجر تطبيقات / منتجات رقمية يدعم الدفع بـ **Pi Network** (π).

## التقنيات
- Backend: Node.js + Express
- Frontend: Vite + (Pug أو HTML/JS)
- Payments: Pi Network User-to-App payments

## المتطلبات
- Node.js ≥ 18
- حساب مطور Pi (https://minepi.com/developers)

1. Clone الريبو  
2. `npm install`
